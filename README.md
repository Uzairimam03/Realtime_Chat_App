# 💬 Realtime Chat App

A modern real-time chat application built using **React.js**, **Tailwind CSS**, and **Socket.IO**. This project demonstrates bi-directional messaging with live updates, responsive UI, and clean code architecture.

---

## 🚀 Tech Stack

- React.js
- Tailwind CSS
- Socket.IO
- Node.js + Express (Backend)
- Vite (Frontend setup)

---

## ✨ Features

- 🔄 Realtime messaging using WebSockets
- 🔔 Instant message updates
- 💬 Clean chat interface
- 📱 Fully responsive design
- 🧑‍🤝‍🧑 Multi-user support

---

## 📦 Installation

### 1. Clone the Repository

```bash
git clone https://github.com/Uzairimam03/Realtime-Chat-App.git
cd Realtime-Chat-App
```
### 2. Frontend
```bash
cd frontend
npm install
```
### 3. Backend
```bash
cd backend
npm install
```
### 4. Run the application 
Start the backend:
```bash
cd backend
npm run dev
```
Start the frontend:
```bash
cd ../frontend
npm run dev
```
